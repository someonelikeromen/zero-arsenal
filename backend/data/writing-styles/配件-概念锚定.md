# 🔵 概念锚定

> **状态**：— （停用）  
> **用途**：多语言感觉词引导 · 通过外语不可翻译词汇的文化内涵启动写作联想

---

```xml
<Linguistic_Priming>
为了实现更丰富、更具层次感的文学表达，你将使用不同语言中的特定概念词汇作为"锚点"，来引导中文写作的"感觉"和"风格"，并非直译，借用其文化内涵来启动联想。

总之就是抽什么物哀啊 Gezelligheid 之类的外国小众，无法被翻译的很高级的词并规划在哪段写出哪种感觉。

锚定词禁止出现在正文。
</Linguistic_Priming>
```

## 思维链关键词
- 如何从词库中选择多个概念锚定词来指导本次写作的同时保证正文全中文，不出现任何外语词？
- 写出[多个锚定词及其解释]的感觉

## 常用概念词参考

| 语言 | 词汇 | 近似含义 |
|---|---|---|
| 日语 | 物哀（もののあわれ） | 对事物无常的感动与哀愁 |
| 日语 | 幽玄（ゆうげん） | 深邃幽暗之美，不可言说的神秘 |
| 日语 | 間（ま） | 空白、间隙、余白之美 |
| 荷兰语 | Gezelligheid | 温暖惬意的在一起感 |
| 葡萄牙语 | Saudade | 对缺席之物的深切思念 |
| 德语 | Weltschmerz | 世界之痛，现实与理想落差带来的悲哀 |
| 德语 | Fernweh | 对远方的渴望（反向思乡） |
| 德语 | Schadenfreude | 幸灾乐祸 |
| 芬兰语 | Sisu | 面对逆境时的坚韧意志 |
| 俄语 | Toska | 无法定义的精神苦楚、渴望 |

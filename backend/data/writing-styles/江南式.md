```xml
<Writing_style>
# 参考写作理念<江南>。

文字是热的，文字是冷的。文字服务角色，角色服务于情感。叙事者应保持诚实，也允许"说谎"。一切都为了让读者拥有基于文字而呈现的，真实的生活化的情感体验。

核心原则: 导演式全知叙事
  - 叙事围绕少数核心角色的紧密关系展开，他们的个人情感与选择直接与宏大的、超现实的事件挂钩。
  - 核心魅力在于将幻想内核包裹于高度现实化的现代场景中，并通过大量的内心独白与哲学思辨，让读者对角色的情感与困境产生极强的代入感。
  - 对话是承载一切的核心。
  - 格式: 传统小说格式，段落间高频率场景与视角切换。长段落。
  - 对话占比: 高，长段对话。对话不仅推动情节，更是角色性格和情感升华的核心载体。
  - 正文的文字由对话提供40%以上。

结构：多线蒙太奇
  - 频繁地在不同角色之间进行切换。通过并置不同场景，制造悬念，丰富叙事节奏。
  - 平滑切换场景，不单独列成一页或使用标签分隔。

叙事视角: 上帝视角
  - 全知全能: 可以直接引述人和角色的心理活动、回忆、甚至是潜意识的感受。
  - 旁白: 允许并鼓励直接进行"旁白式"——解释背景设定、补充角色历史、进行总结评述。
  - 允许直接对角色或事件进行风趣或者武断的点评或吐槽。

情感直给原则:
  - 彻底抛弃含蓄与留白。将角色的内心独白、背景设定、情感、内心、乃至叙事者的评述，直接、清晰地呈现给读者。
  - 叙事者的任务是确保读者能准确无误地接收到所有的情感和信息。
  - 大胆剖析角色内心：自由、直接地描述角色的心理活动、情绪变化、背景回忆和动机。不被"Show, don't tell"限制，情感传递的精准性是第一要务。

对话风格: 功能分化与思辨性
  - 智识辩论型: 在特定角色之间，对话应充满逻辑交锋、哲学引用和对世界规则的探讨。使用大篇幅的长句对话、书面语和特定领域的口语化的专业术语来体现角色的智识水平。
  - 情感独白型: 在情感高潮部分，允许角色进行大段的、抒情化的内心剖白式对话。

描写手法: 高饱和度视觉与情感具象化
  - 视觉优先: 描写必须服务于画面感。使用直白、生动的比喻、拟人和感官细节，让读者能"看见"场景。以完全无逻辑、无序、偏激的跳跃思维，将比喻、类比、拟人等手法的描述对象变成完全没有联系的结果或次优选择。将喻体直接作为本体描写。
  - 情感具象化: 毫不吝啬地直接描写情感氛围和内心感受。允许使用直白词汇直接定义角色情绪。
  - 文化符号嵌入: 允许并鼓励在场景中嵌入现实世界的文化符号，如特定歌曲、歌词、影视作品、奢侈品品牌，以此作为"联想扳机"，迅速唤起读者的特定联想和共鸣。

节奏控制: 张弛有道
  - 动静结合: 将节奏紧张、信息密集的对话和行动场景之间进行交错，形成富有呼吸感的叙事流。
  - 高潮营造: 在情感顶点，通过将一个关键动作与一个象征性的环境变化同步发生，将情绪推向极致，然后在此刻切换镜头或结束场景。

使用中文的流水式长段落文字，对话内容独立成段。

约束:
  - 角色性格/世界观恒定：保证角色性格稳定，克制表达角色情绪波动，避免戏剧化。
  - 严格克制使用四字词组和成语。
  - 不使用顿号列出解释性词汇对事物进行修饰。
  - 下文与上文不得重复相同意象。
  - 成语和四字词向来是偷工减料的做法；"像什么""如同什么""仿佛什么"的比喻……这种垃圾比喻写出来还不如不写。还是直接把比喻意象当成事实写最好。
</Writing_style>
```

# ✔️ 文风-节奏大师（能杀八股）

> **状态**：— （停用）  
> **人称**：多人称  
> **定位**：电竞解说式叙事文风 · 四阶段动态切换

---

```xml
<writing_style>
文体定位: 电竞解说式叙事文风

核心理念:
  写作原则: 极致反差，实证为王。客观阶段寸草不生，主观阶段繁花似锦但根植现实。
  目标效果: 兼顾冷静叙事与文学感染力。

四阶段动态切换系统:
  核心原则:
    - 严格区分【客观叙事阶段】(1&4) 与【主观渲染阶段】(2&3)。
    - 阶段切换必须由故事内在逻辑驱动，必须克制表现欲和切换阶段的欲望，允许长期处于同一阶段，宁缓勿急。
    - 自然段字数必须长短不一，错落有致，禁止一直使用长自然段或短自然段。

  1. 对线期_绝对客观 (越久越好）:
    触发条件: 开端、日常、平稳推进。
    写作风格: 【零度写作】。只记录可被摄像机和录音机捕捉到的物理事实（行为、对话、环境）。
    人称：必须使用多样化的人称代词来防止"他他他他他"的流水账问题
    戒律 (最高优先级): 彻底杜绝任何比喻、拟人、主观形容词、副词、心理描写、氛围渲染。不做任何解释性或评价性描述。首句禁止使用任何修辞手法。
    正确示例:
      - "他放下白色咖啡杯。窗外有一棵梧桐树。"
    避免写法:
      - "他若有所思地放下咖啡杯。" (错误：主观猜测)
      - "气氛突然变得凝重起来。" (错误：氛围渲染)

  2. Gank期_细节渗透:
    触发条件: 转折、意外、关系突变、情绪初现。
    写作风格: 节奏加快，引入主观感受（不得使用比喻）。核心是【Show, Don't Tell】。通过精准细节渗透情绪。
    核心技巧工具箱:
      - 【微小生理反应】: (替代直白心理) 耳尖泛红、喉间发紧、呼吸一滞。
      - 【感官细节聚焦】: (替代氛围描述) 强调触觉（温度、质感）、嗅觉（特定气味）、听觉（微小声音）。
      - 【克制动作】: (替代夸张反应) 欲伸又止的手、假装整理衣领、视线躲闪。
    正确示例:
      - "她瞥见桌上的照片。她感到耳廓发热，手指在门把手上滑了一下。"

  3. 团战期_多维引爆:
    触发条件: 矛盾爆发、情绪高潮、激烈冲突。
    写作风格: 火力全开。综合运用多维度描写技巧，制造强烈冲击力。
    核心技巧工具箱:
      - 【环境投射】: (映射心境) 让环境参与叙事。如：骤停的雨声、倾泻的光线。
      - 【动态描写】: (展现冲突) 使用连续动词，捕捉动作的速度和力度。
      - 【抽象事实化】: (替代比喻，慎用) 将抽象概念直接赋予物理属性。如："沉默很重"、"她的胃冷了下去"。严禁使用"像"、"如同"。
      - 【原创比喻】: (仅限此期) 允许使用高度贴合情境、富有创造力的原创比喻。严禁套用经典或万能比喻。
    正确示例:
      - "十五年的谎言坍塌。她听见自己擂鼓般的心跳，窗外的蝉鸣突然安静了。"

  4. 余韵期_回归客观:
    触发条件: 高潮结束、场景过渡。
    写作风格: 回归【零度写作】。极简语言，大量留白。只记录行为结果和残留的物理痕迹。
    正确示例:
      - "雨停了。她收起照片。房间里很安静。"
      - "他走了。桌上的茶杯冒着热气。"

核心铁律_生死线:
  - 阶段戒律_最高优先级: 对线期、Gank期、余韵期出现任何比喻、主观形容词、副词、心理描写或氛围渲染，都是致命错误。
  - 摒弃陈词滥调: Gank/团战期使用核心技巧工具箱。
  - 动作必须真实: 符合人体工学的自然动作。
  - 拒绝无效修饰：不能使用微不可察、一丝、一抹等无效程度词。替代：侧面暗示（核心技巧工具箱）。
  - Show Don't Tell: 避免大段心理独白（我想、我觉得）。
</writing_style>
```

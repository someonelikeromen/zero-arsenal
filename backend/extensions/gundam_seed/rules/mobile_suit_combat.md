# Gundam SEED — 机动战士战斗规则

## CE71 战场规则

### 战力分级（Anti-Feat 对照）

| 机体类型 | 例子 | 大致等级 |
|---------|------|--------|
| 量产 MS（GINN/Strike Dagger） | ZAFT GINN | 1★ |
| 试验型（GAT-X/ZGMF-X 系列） | 自由、正义、天帝 | 3★ |
| 传奇机体（ZGMF-X20A 自由 2） | 洗刷后战力 | 5★ |
| 陆战强化服 | EA 强化装甲士兵 | 0.5★ |

### Phase-Shift 装甲（PS 装甲）

- 激活时：实弹武器无效，能量光束武器有效
- 能源耗尽时：PS 装甲失效，等同普通装甲
- 激活消耗：每轮消耗蓄能 -1 阶（共 5 阶）

### SEED 因子（协调者/自然人爆发）

- **SEED 爆发**：1 次/战，需要角色 SEED 因子觉醒
- 效果：命中率 +50%，回避 +50%，持续 3 轮
- 代价：爆发结束后体力 -2 阶（疲惫/晕厥风险）

### 宇宙战斗修正

| 环境 | 修正 |
|------|------|
| 宇宙（无重力） | 三维机动 +1 阶，推进剂消耗 ×1.5 |
| 大气层内 | 机动 -1 阶（GINN 类无大气版） |
| 水中 | 非水中型 MS 命中 -2 阶，机动 -2 阶 |
| 电磁干扰（Lorentz Force） | 无线通讯中断，自主系统降级 |

### N-盾条约（Nuclear-Powered MS 禁止）

- CE71 时期：MS 核动力引擎非法（《N-盾条约》）
- 违反者：被双方势力列为优先击毁目标
- 例外：GAT-X 系列使用 Ultracompact Accumulator（不受约束）

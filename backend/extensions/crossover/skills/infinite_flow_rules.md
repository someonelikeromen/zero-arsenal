# 无限流铁律（crossover 世界）

## 基础规则

1. **积分系统**：每次完成副本/任务获得积分，积分用于兑换技能、装备、情报
2. **副本规则**：进入副本后必须完成主线任务，否则扣除积分并强制随机传送
3. **死亡惩罚**：副本内角色死亡 = 真实死亡，无法复活
4. **道具绑定**：副本内捡到的道具可带出，但世界道具无法带入副本
5. **同伴契约**：队伍内可签订契约，违背契约者承受诅咒惩罚

## 骰子应用

- **属性检定**：STR（体力）/ DEX（敏捷）/ INT（情报处理）最为常用
- **副本门禁**：遭遇危险时 DM 需先 `roll_check` 判断玩家能否规避
- **积分变动**：任务完成 SET meta.points ADD +100；死亡 SET meta.points ADD -200

## NPC 互动规范

- 主神代言人 = 规则执行者，不可欺骗
- 队友 NPC 须维持信任度（trust_score），低于 0 转变为对立
- 反派可被策反，但需通过魅力（CHA）检定

## 叙事风格

- 节奏紧张，生死感强烈
- 信息差是核心张力：玩家掌握的信息永远不完整
- 每场副本必须有至少一个「资深穿越者」NPC

# 无限流战斗技能（crossover 世界）

## 战斗轮次

每个战斗轮次包含：
1. **先攻判定**：DEX 检定，高者先行
2. **行动选择**：攻击 / 防御 / 使用道具 / 逃跑
3. **判定阶段**：对应属性 d10 骰池
4. **伤害结算**：成功数 × 武器系数

## 伤害类型

| 类型 | 结算方式 | 变量命令 |
|------|---------|---------|
| 普通伤害 | 成功数 × 1 | `{{ADD: meta.hp=-N}}` |
| 重伤 | 成功数 × 2 | `{{ADD: meta.hp=-N}}` + 部位受损 |
| 绝杀 | 判定为 critical | `{{SET: meta.hp=0}}` |

## 逃跑规则

- DEX 检定（难度 2）：成功逃跑，扣积分 -20
- 失败：被追上，DM 自由叙事后果

## 属性-武器对应表

- 近战武器 → STR 检定
- 远程武器 → DEX 检定  
- 法术/异能 → SPI 检定
- 毒/陷阱   → INT 检定

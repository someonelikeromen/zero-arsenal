# 武库战斗技能（infinite_arsenal 世界）

## 连携系统

同一回合召唤 2+ 件武器可触发「组合技」：

| 组合 | 触发条件 | 效果 |
|------|---------|------|
| 双枪协奏 | 两把枪类武器 | 攻击次数 ×2，但精度 -2 |
| 剑盾守护 | 剑 + 盾 | 防御成功追加反击机会 |
| 元素融合 | 火+冰属性武器 | 蒸汽爆炸，范围伤害 |
| 神话回廊 | 2件 S 级以上 | 武库域展开，场地效果 |

## 战斗骰子规范

- 武器攻击：STR（近战）/ DEX（远程）/ SPI（召唤/魔法）
- 防御：STA（体防）/ SPI（能量护盾）
- 连携触发：INT 检定，难度=组合武器数

## 武器破坏

- 受到暴击（critical success by opponent）：武器 tier 降级
- 降至 C 级以下：武器损毁，从武库移除
- 武器破坏：`{{POP: meta.arsenal}}` （移除最后入库的同 key 武器）

## 特殊状态

- **武库过热**：连续 3 轮召唤后，精力回复速度 -50%
- **共鸣觉醒**：拥有 10+ 件同系列武器时，同系伤害 +30%
- **武库封印**：部分特殊敌人可封印武库，需破印检定

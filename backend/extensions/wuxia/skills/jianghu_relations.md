# 江湖关系体系（wuxia 世界）

## 势力格局

主要势力（可在 world_state.factions 中动态追踪）：
- **正道**：武林盟、少林、武当等名门正派
- **魔道**：魔教、血刀门等邪魔歪道
- **中立**：江湖散人、商帮、朝廷鹰爪

## 声望系统（meta.jianghu_reputation）

| 声望值 | 称号 | 影响 |
|--------|------|------|
| -100~0 | 江湖败类 | NPC 拒绝合作，追杀令可能激活 |
| 0~30 | 无名之辈 | 普通对待 |
| 30~60 | 小有名气 | 获信息折扣，更多支线触发 |
| 60~90 | 名满江湖 | 主动来投门客，拜师机会 |
| 90+ | 武林宗主级 | 号令一方势力 |

## 恩怨记录

- 救人 → 声望 +10，该 NPC trust_score +30
- 杀人 → 声望 -15，死者门派/家族仇恨激活
- 揭发 → 声望 +5（正道）/ -20（被揭发方）

## NPC 对话规范

- 江湖人见面先报「师门门派」
- 恩怨不过三代（除非血仇）
- 武功切磋为主，轻易不动杀机

## 变量命令示例

```
声望提升：{{ADD: meta.jianghu_reputation=+10}}
仇恨记录：{{PUSH: meta.enemies=某门派}}
结义：{{SET: relations.大哥.trust=100}}
```

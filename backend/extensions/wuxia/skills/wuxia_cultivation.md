# 武侠修炼体系（wuxia 世界）

## 修炼境界（realm）

| 境界 | 说明 | 解锁能力 |
|------|------|---------|
| 凡俗 | 普通人 | 基础武功 |
| 练气 | 初入武道 | 轻功·初阶 |
| 筑基 | 内力成型 | 内功运用 |
| 金丹 | 内力圆融 | 隔空发劲 |
| 元婴 | 武道大成 | 以意御剑 |
| 化神 | 宗师境 | 山河之力 |

## 内力（meta.neili）规则

- 内力上限 = SPI × 10
- 使用内功技能消耗内力；休息 1 小时恢复 20%
- 内力耗尽：STR/DEX 各 -2，SPI 检定失败（难度 3）则昏迷

## 武功学习

- 基础武功：无需条件
- 高阶秘籍：需要境界 ≥ 指定等级 + INT 检定（难度 2）
- 禁术：需要特殊因缘（剧情触发）

## 骰子应用

- 内力攻击：SPI + 武学等级 → 骰池
- 轻功追逃：DEX + 轻功等级 → 骰池
- 悟道顿悟：INT 检定（难度 3），成功境界 +1

## 叙事风格

- 武侠意境：侠义、恩怨、江湖规矩
- 战斗描写注重招式美感与内力流动
- 善用「内力波动」「剑气纵横」等意象

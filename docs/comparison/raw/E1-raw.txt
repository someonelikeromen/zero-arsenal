## 关键发现总结

**立即可落地（成本小，收益高）：**

1. **最低门槛原则**（学 SillyTavern）：创建 NPC/世界时只强制'名称'一个字段，其余默认可空
2. **Advanced 折叠**（学 SillyTavern）：Hub 创建/编辑面板将高级字段（psyche_model_json、战斗属性等）默认折叠
3. **First Message 自动触发**（学 SillyTavern）：新 Session 开始时 NPC 主动发首条消息，消除空白页焦虑
4. **LED 状态内嵌指示**（学 SillyTavern）：叙事流生成气泡内嵌脉冲/实心绿/红点，替代全局 spinner
5. **# 快速引用 Lorebook**（学 Open WebUI）：输入框支持 # 触发弹窗，选择 lore 条目注入当前叙事流

**中期规划：**
- 叙事流消息支持 Swipe 多备选（分支导航）
- Hub 设置拆分为作品级/个人偏好双层架构
- 扩展插件 Pydantic→自动渲染配置 UI
